Structura Internshipului

1. Însușirea cunoștintelor legate de Java în Android și JavaScript în React Native, cu ajutorul programatorilor seniori și cursurilor de pe platforma online.

2. Studierea aplicațiilor existente și dezvoltarea unor aplicații noi cu ajutorul cunoștintelor dobândite în stagiul 1.

Acest internship vă va dezvolta următoarele abilități: lucrul în echipă, programare Java în Android și Javascript în React Native, cunoștințele despre tehnologii mobile, dar și concepte de project management.
