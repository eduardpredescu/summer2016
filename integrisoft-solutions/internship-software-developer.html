Descrierea jobului

Realizarea de mini-aplicatii web utilizand T-SQL, .Net Framework 3.5 / 4.5, C# 5.0, Reporting Services, lucrul cu un framework custom (Avantis) dezvoltat peste .NET

Timpul de lucru este de 6 ore pe zi sau 30 ore pe saptamana

Va participa la sedintele, dezbaterile / prezentarile tehnice si worshopurile ce nu implica confidentialitate pe termen mediu si lung
